Curso de Bootstrap

Este é um container fixo.

Em Bootstrap, um "container fixo" se refere à classe .container, que fornece um contêiner responsivo de largura fixa para o conteúdo da página. Isso significa que o contêiner se ajusta a diferentes tamanhos de tela, com larguras máximas específicas para cada ponto de interrupção (breakpoints) do Bootstrap

Este é um container fluído.

Em Bootstrap, container-fluid é uma classe que cria um contêiner com largura total, preenchendo toda a largura da janela de visualização. Diferente do contêiner regular (.container), que tem largura máxima e márgenes, o container-fluid se estende por toda a tela em todos os pontos de interrupção
